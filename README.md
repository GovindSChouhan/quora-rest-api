# Quora REST API

A RESTful CRUD application built using Express.js and EJS.

## Features
- RESTful routing
- Create, Read, Update, Delete (CRUD)
- Dynamic routing using route parameters
- EJS templating
- UUID for unique post IDs
- Express middleware
- Static files (CSS)
- Form handling with `express.urlencoded()`

## Tech Stack
- Node.js
- Express.js
- EJS
- HTML
- CSS
- UUID
